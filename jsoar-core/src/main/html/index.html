This is a jsoar agent running in a Java applet. It requires Java 1.6.

Java 1.6 is required to run this applet.

Many of the standard Soar commands are supported. Note that the source command has limited support for reading productions from URLs. A few commands are pre-loaded into the applet's command entry box

Note that Java memory usage is limited in a browser so many large agents will have issues in the applet. Download the full version of jsoar to get full runtime capabilities.
